# Help Desk Ticketing Lab (Freshdesk)

## Overview
This lab demonstrates a simulated IT help desk environment using Freshdesk. It focuses on ticket intake, prioritization, troubleshooting, escalation, and resolution following real-world IT support workflows.

## Tools Used
- Freshdesk (Free Tier)
- Windows OS
- GitHub

## Skills Demonstrated
- Ticket lifecycle management (Open → Resolved → Closed)
- Priority-based incident handling
- L1 to L2 escalation workflows
- VPN, account access, and basic network troubleshooting
- Internal (private) vs external (public) ticket communication

## Ticket Scenarios
- Password Reset (L1)
- VPN Not Connecting – Escalated to Systems Admin (L2)
- Multiple concurrent ticket management

## Screenshots Overview
